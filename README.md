README

This repository contains code and data for a minimal redundant control system with a nonlinear mapping from a three-dimensional state to a two-dimensional task output.

Simulations examine how perturbation magnitude and control gain affect task error, null-space variability (V_null), and recovery dynamics.

Results show that performance degradation is associated more strongly with local ill-conditioning (Jacobian condition number and proximity to singularity) than with the magnitude of task-equivalent variability.

All scripts reproduce the figures and summary metrics reported in the manuscript.
